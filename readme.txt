Crea e gestisci facilmente le tue FAQ personalizzate su WordPress con una GUI semplice ed intuitiva!

== Description ==

Custom Easy FAQ Manager è un plugin leggero ma potente che ti permette di creare e gestire facilmente le tue FAQ (Frequently Asked Questions) sul tuo sito WordPress.

**Caratteristiche principali:**

* Interfaccia utente semplice e intuitiva
* Effetto accordion personalizzabile
* Blocco Gutenberg dedicato
* Shortcode per inserire le FAQ in qualsiasi pagina
* Possibilità di personalizzare colori e stili
* Supporto per la ricerca nelle FAQ
* Editor visuale integrato
* Compatibilità responsive per tutti i dispositivi

**Come utilizzare il plugin:**

1. Dopo l'installazione, troverai il menu "FAQ Manager" nella dashboard di WordPress
2. Aggiungi le tue FAQ con domande e risposte
3. Inserisci le FAQ nelle tue pagine tramite il blocco Gutenberg dedicato o lo shortcode [custom_faq]
4. Personalizza l'aspetto grafico tramite le impostazioni del plugin

== Installation ==

1. Carica i file del plugin nella cartella `/wp-content/plugins/`
2. Attiva il plugin attraverso il menu 'Plugin' in WordPress
3. Configura il plugin e crea le tue FAQ dal menu "FAQ Manager"
4. Usa il blocco Gutenberg "FAQ" o lo shortcode [custom_faq] per visualizzare le FAQ nelle tue pagine

== Frequently Asked Questions ==

= Come posso inserire le FAQ in una pagina? =

Puoi utilizzare il blocco Gutenberg "FAQ" o inserire lo shortcode [custom_faq] in qualsiasi editor di pagina o post.

= Posso personalizzare i colori delle FAQ? =

Sì, dalle impostazioni del plugin puoi personalizzare i colori di sfondo e del testo per domande e risposte.

= È possibile visualizzare solo alcune FAQ specifiche? =

Sì, puoi utilizzare lo shortcode con parametri aggiuntivi: [custom_faq id="1,2,3"] per mostrare solo le FAQ con quegli ID.

= Il plugin è compatibile con i temi WordPress moderni? =

Sì, il plugin è stato progettato per funzionare con qualsiasi tema WordPress moderno e supporta la modalità responsive.

== Privacy Policy ==

Custom Easy FAQ Manager non raccoglie né condivide dati personali degli utenti. Tutti i dati inseriti tramite il plugin vengono archiviati solo nel database di WordPress e non vengono mai trasmessi a servizi esterni.
